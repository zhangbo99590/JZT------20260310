# AI政策中心功能说明

## 概述

AI政策中心是企规宝系统的核心模块之一，提供智能化的政策查询、匹配、申报辅助等功能。通过AI技术，为企业提供个性化的政策服务。

## 功能模块

### 一、顶部AI交互核心模块

#### 1. 自然语言AI搜索栏
- **功能**：支持用户以日常语言提问，AI通过语义理解解析需求
- **示例**："北京小型科技企业2024年能领的补贴政策有哪些？"
- **特点**：
  - 实时显示AI解析进度
  - 智能语义匹配，非单纯关键词搜索
  - 搜索历史记录（最近10条）

#### 2. AI智能联想推荐
- **功能**：根据用户输入自动推荐智能提问模板
- **示例**：输入"补贴" → 推荐"XX地区XX行业补贴政策申报条件"
- **特点**：
  - 实时联想
  - 一键填充
  - 减少输入成本

#### 3. 语音输入转AI提问
- **功能**：支持语音输入，自动转文字并解析
- **特点**：
  - 支持普通话识别
  - 降低操作门槛
  - 适配不同使用习惯

#### 4. 企业画像绑定与管理
- **功能**：绑定企业核心信息，实现千人千面服务
- **信息包括**：
  - 企业名称、所在地区
  - 行业、规模
  - 核心资质（高新技术企业、专精特新等）
  - 注册资本、员工数、年营收
- **特点**：
  - 支持编辑和切换
  - 所有AI功能基于企业画像提供个性化服务

### 二、中间AI政策服务核心模块

#### 1. AI精准匹配政策页
- **AI推荐政策卡片**：
  - 匹配度标识（颜色+百分比）
    - 红色：≥80%（高匹配）
    - 黄色：60-79%（中匹配）
    - 灰色：<60%（低匹配）
  - 匹配理由说明
  - 核心信息展示
  - 申报截止倒计时
  - 一键操作（AI解读、适配性判断）

- **换一批AI推荐**：
  - 调整推荐逻辑
  - 支持回溯上一批

#### 2. AI实时申报适配性判断
- **功能**：点击"我能申报吗？"实时分析
- **输出结果**：
  - 适配结论（符合/部分符合/不符合）
  - 通过概率
  - 改进建议
  - 缺失条件说明
  - 类似可申报政策推荐

#### 3. AI搜索结果智能解析页
- **AI语义聚合分类**：
  - 按政策类型分类
  - 按申报紧急度分类
  - 按适配企业规模分类

- **AI政策摘要卡片**：
  - 申报条件（硬性/软性）
  - 所需材料（标注可AI生成/需盖章/需第三方认证）
  - 补贴金额（最高/最低/平均）
  - 截止时间（实时倒计时）

- **AI多政策对比**：
  - 支持选择2-5个政策
  - 横向对比表格
  - 对比维度：条件、材料、金额、难度、截止时间、适配度
  - 支持导出Excel

#### 4. AI政策辅助工具（右侧固定）
- **AI材料生成**：
  - 基于企业画像自动生成申报材料草稿
  - 支持在线编辑
  - 导出Word/PDF

- **AI进度跟踪**：
  - 绑定申报项目
  - 实时监控审批节点
  - 节点更新推送提醒
  - 显示预计下一步时间

- **AI政策订阅**：
  - 自定义订阅维度（地区+行业+类别）
  - 新政策发布自动推送
  - 政策变更提醒
  - 支持多种接收方式（系统消息/短信/邮件）

### 三、底部AI政策洞察模块

#### 1. 企业政策机会数据卡片
- **展示数据**：
  - 可申报政策数
  - 预估可获补贴总额
  - 高适配政策数（匹配度≥80%）
- **特点**：
  - 可视化卡片展示
  - 每日自动更新
  - 标注更新时间

#### 2. AI申报优先级建议
- **排序依据**：
  - 补贴金额
  - 申报难度
  - 截止时间
- **建议分类**：
  - 优先申报（高补贴+低难度+即将截止）
  - 次优先申报（中等条件）
  - 长期关注（无截止时间/需补充资质）
- **特点**：
  - 点击直接跳转详情
  - 缩短决策链路

#### 3. AI资质缺口提醒与指引
- **功能**：
  - 对比企业现有资质与政策要求
  - 标红提醒缺口
  - 推荐资质获取路径
- **示例**：
  - 缺口：发明专利≥2项（当前仅1项）
  - 建议：申报"XX专利补贴政策"
  - 指引：查看资质获取指南

#### 4. 数据可视化
- **政策类别分布图**：饼图展示不同类别政策占比
- **补贴金额趋势图**：柱状图展示预估补贴趋势

## 使用说明

### 1. 首次使用
1. 点击搜索栏右侧的"绑定企业信息"按钮
2. 填写企业基本信息和资质
3. 保存后，AI将基于企业画像推荐政策

### 2. 搜索政策
- 方式一：在搜索栏输入自然语言问题
- 方式二：点击智能提问模板
- 方式三：使用语音输入

### 3. 查看推荐
- 查看AI推荐的政策卡片
- 点击"AI解读"了解政策详情
- 点击"我能申报吗？"检查适配性

### 4. 对比政策
1. 勾选2-5个政策
2. 点击"AI对比分析"
3. 查看横向对比表格

### 5. 申报辅助
- 使用AI材料生成功能
- 添加申报进度跟踪
- 订阅相关政策更新

### 6. 查看洞察
- 查看企业政策机会数据
- 参考AI申报优先级建议
- 关注资质缺口提醒

## 注意事项

1. **企业画像**：完善的企业信息有助于AI更精准推荐
2. **数据更新**：洞察数据每日凌晨自动更新
3. **语音输入**：需要浏览器支持Web Speech API
4. **材料生成**：AI生成的材料仅供参考，需人工审核
5. **进度跟踪**：需手动输入申报编号
