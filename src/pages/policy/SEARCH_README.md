# AI政策搜索页面说明

## 概述

这是一个简化的AI政策搜索页面，用户输入搜索内容后，系统通过AI匹配展示相关政策结果。

## 页面结构

### 1. 搜索区域（顶部）
- **渐变背景**：紫色渐变设计，视觉吸引力强
- **大标题**："AI政策智能搜索"
- **副标题**：引导用户使用自然语言描述需求
- **搜索框**：
  - 支持自然语言输入
  - 示例占位符："例如：北京市丰台区高新技术企业复审给多钱，再附上相关政策"
  - AI搜索按钮
- **热门搜索标签**：
  - 高新技术企业补贴
  - 专精特新认定
  - 研发费用补贴
  - 点击标签自动填充并搜索

### 2. 搜索结果区域（主体）
- **结果统计**：显示找到的政策数量
- **政策卡片列表**：
  - 每个卡片包含：
    - 热门标识（红色Badge）
    - 政策标题
    - 匹配度百分比（颜色编码）
      - 红色：90%以上
      - 橙色：80-89%
      - 绿色：70-79%
      - 蓝色：70%以下
    - 政策摘要
    - 标签（类型、地区、级别、补贴金额、截止时间）
    - 来源数量和来源标签
    - 操作按钮（查看详情、我能申报吗？）

### 3. 政策详情抽屉（右侧滑出）
点击政策卡片后展示详细信息：
- **基本信息**：标签、地区、级别、补贴金额
- **政策概述**：政策的基本介绍
- **奖励金额说明**：详细的补贴标准
- **主要政策依据**：政策文件来源
- **实务操作提醒**：申报流程和注意事项
- **相关政策**：关联政策链接
- **来源**：所有信息来源标签

## 技术实现

### 组件结构
```
AIPolicySearch.tsx          # 主组件
AIPolicySearch.module.css   # 样式文件
```

### 数据结构
```typescript
interface PolicyResult {
  id: string;                    // 政策ID
  title: string;                 // 政策标题
  summary: string;               // 政策摘要
  tags: string[];                // 标签
  level: string;                 // 级别
  region: string;                // 地区
  deadline?: string;             // 截止时间
  subsidy?: string;              // 补贴金额
  matchScore: number;            // 匹配度
  isHot?: boolean;               // 是否热门
  sources: PolicySource[];       // 来源
  content: PolicyContent;        // 详细内容
}
```

### 核心功能
1. **搜索功能**：`handleSearch()`
   - 接收用户输入
   - 调用AI匹配（模拟）
   - 展示搜索结果

2. **详情展示**：`handleViewDetail()`
   - 打开右侧抽屉
   - 展示完整政策信息

3. **匹配度计算**：`getMatchScoreColor()`
   - 根据分数返回对应颜色
   - 视觉化匹配程度

## 使用方法

### 1. 基本搜索
1. 在搜索框输入问题（如："北京市高新技术企业补贴"）
2. 点击"AI搜索"按钮或按回车
3. 等待AI分析（约1.5秒）
4. 查看搜索结果

### 2. 快速搜索
1. 点击"热门搜索"下的任意标签
2. 自动填充并执行搜索

### 3. 查看详情
1. 点击任意政策卡片
2. 右侧滑出详情抽屉
3. 查看完整政策信息
4. 点击关闭按钮或遮罩层关闭抽屉

### 4. 适配性判断
1. 点击政策卡片上的"我能申报吗？"按钮
2. 系统分析企业是否符合申报条件
3. 展示分析结果和建议
