# 申报资质类型选择器 - 抽屉式交互优化文档

**优化日期**: 2026-03-04  
**优化类型**: 从下拉式重构为抽屉式交互  
**优化模块**: 申报向导 - 资质类型选择

---

## 一、优化概述

将原有的下拉式资质选择器重构为**抽屉式（Drawer）交互模式**，解决了下拉框空间受限、操作路径长、信息展示不充分等问题，通过"左侧分组导航 + 右侧选项列表"的双栏布局，大幅提升了用户的选择效率和操作体验。

### 3.2 状态管理

```typescript

const [searchText, setSearchText] = useState('');
const [tempSelected, setTempSelected] = useState<string[]>(value);

```

**状态说明**:
- `selectedGroup`: 当前选中的分组
- `searchText`: 搜索关键词
- `tempSelected`: 临时选择状态（确认前）
- `expandedDetailId`: 展开详情的资质ID

### 3.3 数据过滤逻辑

```typescript
const filteredQualifications = useMemo(() => {
  // 高频资质分组
  if (selectedGroup === 'high_frequency') {
    if (!searchText) return highFrequencyQualifications;
    const lowerSearch = searchText.toLowerCase();
    return highFrequencyQualifications.filter(q =>
      q.label.toLowerCase().includes(lowerSearch) ||
      q.pinyin?.includes(lowerSearch) ||
      q.description.toLowerCase().includes(lowerSearch)
    );
  }

  // 其他分组
  const group = QUALIFICATION_DATA.find(g => g.key === selectedGroup);
  if (!group) return [];

  if (!searchText) return group.qualifications;

  const lowerSearch = searchText.toLowerCase();
  return group.qualifications.filter(q =>
    q.label.toLowerCase().includes(lowerSearch) ||
    q.pinyin?.includes(lowerSearch) ||
    q.description.toLowerCase().includes(lowerSearch)
  );
}, [selectedGroup, searchText]);
```

### 3.4 组件接口

```typescript
interface QualificationDrawerProps {

  value?: string[];           // 当前已选资质
  onClose: () => void;        // 关闭回调
  onConfirm: (selectedValues: string[]) => void;  // 确认回调
}
```

## 五、用户操作流程

### 5.1 基本选择流程

1. **打开抽屉**
   - 点击"请选择申报资质类型"输入框
   - 抽屉从右侧滑出

2. **浏览资质**
   - 默认显示"高频资质（推荐）"分组
   - 点击左侧导航切换分组

3. **搜索定位**
   - 在顶部搜索框输入关键词
   - 实时过滤当前分组的资质

4. **查看详情**
   - 点击资质卡片右侧的 ℹ️ 图标
   - 展开查看适用行业、申报条件、核心价值

5. **选择资质**
   - 点击资质卡片或复选框进行选择
   - 已选资质显示蓝色边框和背景
   - 底部已选区域实时更新

6. **确认选择**
   - 点击"确认选择"按钮
   - 抽屉关闭，选择结果回填到表单

### 5.2 快捷操作

- **清空已选**: 点击左下角"清空已选"按钮
- **删除单个**: 点击底部标签的 × 图标
- **取消操作**: 点击"取消"按钮或抽屉外部区域
